# Reference Documentation

This folder contains **factual reference materials** for Amalgam: specifications, compatibility data, and technical reference tables. Use this when looking up specs, verifying printer compatibility, or understanding component requirements.

## Contents

- **donor-printer-guide.md** — Comprehensive guide to donor printer compatibility
  - Fitness tiers (Tier 1/2/3) for each printer model
  - What components are scavengeable from each donor
  - Frame type, bed size, Z-travel specs
  - Mixed-donor combinations and recommendations
  - Cost reality check: when NOT to scavenge
  - Start here if selecting donor printers

- **parts-sourcing-guide.md** — How to source missing/failed parts
  - How to spot counterfeits and duds on AliExpress/Temu/eBay
  - Common pitfalls (wrong pitch leadscrew, fake bearings, etc.)
  - Platform comparison and sourcing strategy
  - Budget planning and decision tree
  - Use this if a donor part fails during build

- **MASTER_PARTS_LIST.md** — Complete bill of materials for all configurations
  - By frame path (Scaffold, Mill, Lathe)
  - By electronics configuration (Dual-MCU, SKIPR)
  - Part sourcing notes and costs

- **RENAME-TRACKING.md** — Project rename history
  - Tracks Neo-Darwin → Amalgam renaming
  - Useful for cross-referencing old documentation

- **Assembly_Manual_Trident.pdf** — Reference material from Voron Trident
  - Not Amalgam-specific, but useful for understanding Z-drop architecture
  - Reference only; see guides/ for Amalgam assembly instructions
